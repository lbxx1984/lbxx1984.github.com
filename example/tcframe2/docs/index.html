TcFrame v2.0 API Docs
简介
    TcFrame v2 是一款非常优秀的轻量级Javascript嵌入式框架，定义了JS的面向对象扩展，不足200KB的基础包中，重新定义了丰富的可视组件及基础特效组件，整体结构清晰明了，文档齐全，API丰富。
    该框架结构风格类似于ActionScript3，并将Flex SDK4.6中很多控件以JS实现，同时获得了很好的浏览器兼容。如果开发者对ActionScript3、JAVA等比较熟悉，会轻松上手，并像使用这些语言一样使用TcFrame v2。
    TcFrame v2重新定义了Web开发的含义，从传统基于页面跳转、依赖URL的Web开发思想迁移到面相对象编程思想，整个Web可以只有一个html文件，而所有功能都由JS来完成。我们基于AJAX提供了丰富的后台接口，开发者可以任选后台技术如php、jsp、.net为前端提供服务支持。用户在访问我们的Web时，会感觉在使用一款常见的PC软件。这样做的好处一是可以给用户提供全新的体验，二是可以大大减轻服务器负担，因为我们开发好的站点，已经保存在用户本地。
    TcFrame v2尤其适用于企业级基于BS模式管理系统的开发，能满足各种管理系统的开发要求，并提供了丰富的Chart图表组件。针对特定用户群体，甚至可以规定浏览器版本，来使用异常强大的WebGL，显示美轮美奂的3D统计图表。当然，它同样适用于常规站点的开发，将TcFrame作为一个小组件，嵌入到网页任何div容器中，这种思想，同制作一个flash嵌入进网页是完全相同的。
    该框架致力于在主流浏览器中保持显示效果一致，当前全面支持IE7、Chrome15、Safari4、Opera10、FireFox16或更高版本浏览器。 在不支持HTML5的浏览器如IE8、IE7中，暂时无法使用Chart图表组件；在不支持WebGL的浏览器中，暂时无法使用3D图表组件。未来，我们会开发出Chart图表和3D图表的flash版及sliverlight版，以弥补统计图表的兼容性不足。
    选择TcFrame v2，将大大提高开发人员的工作效率，将显著降低Web开发的入门等级。TcFrame v2是各种基于浏览器应用的首选开发框架，我们的团队也会是您默契的合作伙伴。
